#1.Primitive Data Type:-
They are immutable (can not be changed, only replaced) & are stored directly in stack memory.
String (stored as utf-16 codes unit),
Number (All Number are 64-bits Floats),
Boolean,
undfined,
Null (type of null === object most importaant bug ofjs)
BigInt,
Symbol

#2.Refrence Type:-They are number & can be complex.they are stored in the heap memory & the variable in the stack only hold pointer(refrence) of that memory location.
Object:- Key:Value
Array:- index:value,
Function:- First Class Citizen,
Date & RegEx:- Specilized object type.
